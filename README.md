# Dito

**Dito** is a futuristic, touch-based web application designed to assign random outcomes using multiple fingers on a touchscreen.
It is a **fully static web app**, runs directly in modern browsers, and requires **no backend or installation**.

👉 **[🎮 Live demo](https://moorada.github.io/dito)**
---

## Features

Dito supports three interactive modes:

### 🔀 Random Selection
Place multiple fingers on the screen and let Dito randomly select a configurable number of them after a short countdown.
Perfect for quick decisions like *“who starts?”* or *“who is chosen?”*.

### 👥 Teams
All active fingers are automatically divided into a chosen number of teams.
Each team is visually distinguished with a futuristic neon color and letter.

### 🔢 Numbers
Each finger receives a unique number after a short countdown.
Ideal for ordering players or assigning positions randomly.

## Technology

- Pure HTML, CSS, and JavaScript
- Canvas-based rendering
- Fully static — works offline once loaded
- Compatible with modern desktop and mobile browsers
